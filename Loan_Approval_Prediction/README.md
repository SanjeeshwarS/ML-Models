## 🚀 Projects List

### 1. Loan Approval Prediction
*   **Description**: Predicts whether a loan will be approved based on applicant details (Income, Credit Score, etc.).
*   **Model**: Logistic Regression
*   **Accuracy**: 79%
*   **Location**: `Loan_Approval_Prediction/`

---

## 🛠️ General Setup and Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SanjeeshwarS/Loan-approval-prediction.git
   cd Loan-approval-prediction
   ```

2. **Create and Activate Virtual Environment**:
   ```powershell
   # Create environment
   python -m venv .venv

   # Activate on Windows
   .\.venv\Scripts\activate

   # Activate on Mac/Linux
   source .venv/bin/activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install pandas scikit-learn
   ```

## 📂 How to run a model
Navigate to the specific project folder and run the `main.py` file:
```bash
cd Loan_Approval_Prediction
python main.py
```
